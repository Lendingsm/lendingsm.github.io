使用腾讯云Lighthouse搭建Lsky-Pro图床

2021-05-09 663 字
信息技术

​ Lsky-Pro（兰空图床）是一款安装简单，功能强大的图床程序。有时候我们会不满意第三方图床，我们就可以利用兰空来自建图床。（这里使用腾讯云轻量应用服务器来进行演示）

1.Lsky-Pro的特点

​ \支持第三方云存储（本地、阿里云OSS、腾讯云COS等）

​ \支持多图上传、拖拽上传、粘贴上传、上传预览、全屏预览

​ \页面响应式布局

​ \全局配置用户存储空间，指定用户剩余存储空间

​ \接口上传、图片软删除

​ \OTA平滑升级

​ \暗黑主题

​ \IP封禁

​ \单用户模式

​ \图片广场

1.简介

​ 项目：https://github.com/wisp-x/lsky-pro

​ 演示：https://pic.iqy.ink/

​ 使用手册：https://www.kancloud.cn/wispx/lsky-pro/880042

2.搭建前的准备

​ \一台安装了宝塔面板的服务器

​ \PHP版本大于等于5.6

​ \Mysql版本大于等于5.5

​ \Nginx1.18版本

​ \PDO拓展

​ \ZipArchive支持

​ \Fileinfo拓展

​ \Curl拓展

（下面教大家如何配置一台装了宝塔面板的服务器）

​ 1.（为了方便我们这里选用腾讯云清凉）选购一台服务器（地域选中国香港，应用镜像选择宝塔面板腾讯云专享版即可）

​ *（付款后等待服务器创建完成，之后进入宝塔面板配置安装环境）

​ 2.这里LNMP安装如上图即可（为了节省资源安装Mysql5.5版本）

​ 3.LNMP安装完成后进入软件商店。

​

*（点击箭头指出来的按钮）

*（之后选择已安装）

*（在下面的列表里找到你安装的PHP，选择设置）

*（选择安装拓展）

*（安装Fileinfo拓展）

3.开始搭建

​ 1.新建一个网站

*（点选添加站点）

*（输入你的域名）

​

*（数据库可以顺便创建）

​

这边数据库密码和账号自己填即可（一定要记住）

2.下载图床程序

​ *（进入你刚创建好的网站的文件夹）

​

https://github.com/wisp-x/lsky-pro/archive/refs/heads/master.zip （源码下载链接）

*（此处利用宝塔的远程下载功能来拉取文件）

*（下载成功后我们点击解压）

*（记得将解压后的文件移动到网站根目录下）

进入网站设置，进行一些配置。

2.设置伪静态。

​ Apache 直接使用程序自带的 .htaccess 即可，nginx需要配置下伪静态，代码如下：

location / {
​ if (!-e $request_filename) {
​ rewrite ^(.*)$ /index.php?s=$1 last; break;
​ }
}
